## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content with your own information

## Customization

### Personal Information

1. **Name**: Replace "Your Name" throughout the HTML file
2. **Title/Subtitle**: Update the hero section subtitle
3. **About Section**: Edit the about text in the `#about` section
4. **Skills**: Modify the skill tags in the `#skills` section
5. **Projects**: Update project cards in the `#projects` section
6. **Contact**: Update email, LinkedIn, and GitHub links in the `#contact` section

### Styling

- Colors can be customized in the `:root` CSS variables at the top of `styles.css`
- Fonts can be changed by updating the Google Fonts link in `index.html`
- Layout spacing can be adjusted in the CSS file

### Contact Form

The contact form is currently set up to show an alert on submission. To make it functional:

1. Use a service like [Formspree](https://formspree.io/) or [EmailJS](https://www.emailjs.com/)
2. Set up a backend API endpoint
3. Update the form submission handler in `script.js`
